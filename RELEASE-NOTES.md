# Release Notes (2026-04-23 to 2026-04-27)

## Added Features

- Replaced the old dashboard "alerts" with a persisted task dashboard (task cards + task detail/form flow).
- Added workflow routing from dashboard tasks (jump straight into the right form to complete the task).
- Auto-generates pregnancy-check follow-up tasks from breeding flows.
- Added sticky follow-up actions in workflows to make completing next steps faster.
- Added breeding record timestamps (record time-of-service, not just the date).
- Added daily log flush follow-up support (capture flush-related follow-up and navigation).
- Reworked "Add Collection" into a step-by-step wizard for better scrollability, including a Motility (%) input on Step 1.
- Support multiple daily checks per mare per day.
- Added a configurable 12h/24h clock setting.
- Added a breeding event detail view.
- Form polish: removed "Optional" placeholders and reduced some routing duplication/fragility.
